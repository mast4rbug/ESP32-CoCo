# Contributing

This is a community documentation project for ESP32-COCO. It started as a scaffold with `TODO` markers because very little about this device is documented publicly (no official GitHub repo, site, or manual as of July 2026) — help filling in real, verified information is welcome.

## How to help

1. Pick a page with `TODO` markers.
2. Replace TODOs with verified information — from your own hands-on testing, the creator, or the CoCo Facebook community.
3. If information is unconfirmed/secondhand, say so explicitly (e.g. "unconfirmed, reported by user X") rather than stating it as fact.
4. Keep cross-references between pages working — update links if you rename or move a file.
5. Log notable changes in [CHANGELOG.md](CHANGELOG.md).

## Style notes

- Prefer short, verified statements over speculation.
- Use tables for reference data (shortcuts, compatibility lists, settings).
- Keep each page focused on one topic; link to related pages rather than duplicating content.
- Mark anything unverified clearly until it's confirmed.
