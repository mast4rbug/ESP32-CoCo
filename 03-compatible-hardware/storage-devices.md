# Storage Devices

## SD cards

- **Standard SD card, FAT32 required.**
- Recommended formatting tool if you hit issues: [SD Card Formatter](https://www.sdcard.org/downloads/formatter/) (SD Association).
- Mac users: strip Finder `._` files before use with [Remove Dot Files](https://maccrafters.com/apps/Remove-Dot-Files.zip) — see [sd-card.md](../01-getting-started/sd-card.md).
