# Glossary

**CoCo** — Common nickname for the Tandy/Radio Shack TRS-80 Color Computer line.

**CoCo 1 / CoCo 2 / CoCo 3** — First-, second-, and third-generation models of the Color Computer. The difference between CoCo 1 and CoCo 2 was minimal (mostly cosmetic/semantic). CoCo 3 added the GIME chip for enhanced video/memory features.

**DECB** — Disk Extended Color BASIC, the disk-based ROM operating environment for the CoCo.

**DSK** — File extension/format for a raw disk image of a CoCo floppy disk.

**CAS** — File extension/format for a raw cassette data image.

**WAV** — Standard audio file format, sometimes used to store an actual audio recording of a CoCo cassette signal for playback into the emulator.

**GIME** — Graphics Interrupt Memory Enhancement chip, the custom chip in the CoCo 3 responsible for enhanced video modes and memory management.

**Pak / Program Pak** — CoCo cartridge format; ROM-based software that auto-runs on insertion.

**Multi-Pak Interface (MPI)** — Original Tandy accessory allowing multiple Program Paks to be connected simultaneously.

**ESP32-S3** — The Espressif microcontroller SoC that ESP32-COCO's hardware emulation runs on.

**OSD** — On-screen display; refers to the device's on-screen menu system.
